# Zoocontrol_Project

Sistema de gestión de zoológico desarrollado en Python utilizando Programación Orientada a Objetos (POO).

Este proyecto incluye clases para representar animales (mamíferos y reptiles), calcular sus raciones diarias de alimento y gestionar un inventario de alimentos.

## Funcionalidades

- Definición de clases para diferentes tipos de animales con herencia y polimorfismo.
- Cálculo automático de la ración diaria de alimento según el tipo y características del animal.
- Gestión de inventario para controlar el stock de alimentos disponible.
- Ejemplos de uso para probar el sistema.

## Tecnologías

- Python 3.x
- Programación Orientada a Objetos (POO)
